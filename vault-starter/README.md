# Vault Starter

This folder contains starter markdown files for the Chief of Staff vault schema.

## Usage
1. Copy the contents of this folder into your vault root (the path in OBSIDIAN_VAULT_PATH).
2. Keep `_kate` files user-owned and editable.
3. Let automated workflows append to `Daily/` and `Meetings/`.

## Included
- `_kate` preference and operating files
- `_kate/templates` starter templates
- empty operational folders for daily and meeting outputs
